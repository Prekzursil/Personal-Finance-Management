# Product Context: Thrifty Personal Finance App

## Goal
Thrifty is an Android application designed to help users manage their personal finances effectively.

## Problem Solved
The app addresses the common challenges of tracking income and expenses, setting financial goals (budgets), and remembering important financial dates or tasks. It aims to consolidate monetary information and provide clear insights into spending habits.

## Core Functionality
- **User Accounts:** Secure signup and login.
- **Income/Expense Tracking:** Users can manually input income and expenses. Expenses can also be added by scanning bills using OCR. Expenses are categorized using tags.
- **Budgeting:** Users can set a monthly budget.
- **Alerts & Reminders:**
    - Notifications are sent if spending exceeds budget thresholds.
    - Users can set custom reminders for specific dates and times.
- **Transaction Viewing:**
    - Dashboard shows a summary (total income/expense, recent transactions).
    - A dedicated screen shows a full list of transactions.
    - Pie charts visualize expenses by tag (likely on the dedicated transaction screen) and income vs. expense (on the dashboard).
- **Data Backup:** Google Drive integration for backup and restore.

## Target User
Individuals looking for a simple mobile tool to monitor their spending, stay within budget, and manage financial tasks.
